# Live Updates

Dartsnut Agent uses `electron-updater` with the `electron-builder` generic provider.
The update feed is static and is expected at:

```text
https://dartsnutstore.oss-cn-hongkong.aliyuncs.com/agent-update/
```

No dedicated update server is required. The host only needs to serve the files
generated by `electron-builder` over HTTPS.

## Required Artifacts

Upload the generated release metadata and binaries without renaming them.

macOS:

- `latest-mac.yml`
- `*.zip`
- `*.zip.blockmap`
- `*.dmg` and `*.dmg.blockmap` for manual download

Windows:

- `latest.yml`
- `*.exe`
- `*.exe.blockmap`

## Cache Headers

- `latest.yml` and `latest-mac.yml`: no-cache or very short TTL.
- Versioned binaries and blockmaps: long immutable cache is fine.

## Packaging Notes

- macOS updates require signed packaged builds. The zip target is included for
  updater compatibility; the dmg can remain the manual installer.
- Windows live updates use the NSIS target. Portable Windows builds can still be
  produced with `pnpm --dir apps/desktop run package:win:portable`, but portable
  builds are not the standard auto-update path.
